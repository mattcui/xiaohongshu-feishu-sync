# 小红书数据抓取

## 描述

本技能用于自动爬取小红书平台的数据，并将结果写入飞书表格。支持自定义搜索关键词、排序方式和抓取数量。

## 使用场景

当需要获取小红书平台上特定关键词的笔记数据，并将其整理到飞书表格中进行分析时使用。

## 首次使用流程

### 步骤1：安装依赖
```bash
cd /Users/mattcui/Downloads/workspace/sync_feishu
./install.sh
```

### 步骤2：配置环境
安装完成后会自动启动配置向导，按提示输入：
- 飞书 APP ID
- 飞书 APP Secret  
- 飞书表格 URL

如需重新配置：
```bash
uv run python3 configure.py
```

### 步骤3：开始抓取
```bash
./run.sh --keys <关键词> --sort <排序方式> --count <数量>
```

## 配置要求

1. **环境变量配置**：在项目根目录创建 `.env` 文件，包含以下内容：
   - `FEISHU_APP_ID`: 飞书应用的APP ID
   - `FEISHU_APP_SECRET`: 飞书应用的APP Secret
   - `BITABLE_URL`: 目标飞书表格的完整URL

2. **依赖安装**：确保已安装所需依赖，运行 `./install.sh`

## 执行步骤

1. 打开终端，进入项目目录：`cd /Users/mattcui/Downloads/workspace/sync_feishu`
2. 运行命令：`./run.sh --keys <关键词> --sort <排序方式> --count <数量>`

## 参数说明

| 参数 | 说明 | 可选值 | 默认值 |
|------|------|--------|--------|
| `--keys` | 搜索关键词，多个用逗号分隔 | 任意关键词 | 必填 |
| `--sort` | 排序方式 | general, popular, latest | general |
| `--count` | 抓取数量 | 1-100 | 15 |

## 示例

```bash
# 抓取针灸相关笔记，按最多点赞排序，抓取10条
./run.sh --keys 针灸 --sort popular --count 10

# 抓取多个关键词，按最新发布排序，抓取20条
./run.sh --keys 针灸,中医美容 --sort latest --count 20
```

## 输出结果

执行完成后，数据将自动写入指定的飞书表格，包含以下字段：
- 笔记ID、笔记标题、笔记链接、笔记类型、笔记正文
- 视频链接、封面图、图片列表、标签
- 发布时间、更新时间、采集时间
- 用户ID、账号昵称、地理位置
- 点赞数、收藏数、评论数、分享数
- 搜索关键词、视频逐字稿

## 注意事项

1. 首次运行需要扫码登录小红书账号
2. 确保飞书应用已正确配置权限
3. 抓取数量建议不要超过100条，避免触发平台限制
